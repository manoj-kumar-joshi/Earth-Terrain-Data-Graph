Overview

This is a simple mobile application like user interface to TG query "tg_walkable_trail_to_nearest_road". This query searches for the smoothest shortest path to the nearest documented road or a trail. Application using this query may be useful to anyone in the mountain or uncharted terrain to find a way home via nearest road. You can click to this Video link to learn more about this.

How To Use ?
1. How to use ?

Drag and drop the location marker (Bubble shape) to simulate your movement. After you relesae the mouse, it will get smoothest shortest path from Tigergraph and render as a colored hexagons. Each hexagon is roughly 10 meters side length.

2. What Does Color Represent In Path ?

Color Band Represent angle of elevation or slope of terrain from green as "lowest" to red as "highest".

3. Street And Satellite Button ?

By clicking Street and Satellite button you can change view of map to street and Satellite view respectively.
